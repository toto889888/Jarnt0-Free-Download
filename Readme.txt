ໂປຣເຈັກ ຈານໂຕ່ດາວໂຫລດວິດີໂອຟຣີ (Jarnt0-Downloads)
ສະບາຍດີທຸກຄົນ! ຍິນດີຕ້ອນຮັບສູ່ໂປຣເຈັກ "ຈານໂຕ່ດາວໂຫລດວິດີໂອຟຣີ" ຂອງຂ້ອຍເດີ້!
ໂປຣເຈັກນີ້ແມ່ນຫຍັງ?
ເວົ້າແບບງ່າຍໆ ເລີຍ, ມັນຄືເຄື່ອງມືນ້ອຍໆ ແຕ່ມີປະໂຫຍດຫຼາຍ ທີ່ຊ່ວຍໃຫ້ເຮົາດາວໂຫລດວິດີໂອຈາກເວັບໄຊຕ່າງໆ ໄດ້ແບບສະບາຍໆ, ບໍ່ວ່າຈະເປັນ YouTube, Facebook, TikTok, ຫຼືແມ້ກະທັ້ງເວັບໄຊອື່ນໆ ອີກຫຼາຍບ່ອນ. ຄິດພາບວ່າເຮົາໄປເຫັນວິດີໂອເດັດໆ ແລ້ວຢາກເກັບໄວ້ເບິ່ງຕອນບໍ່ມີອິນເຕີເນັດ ຫຼື ຢາກສົ່ງໃຫ້ໝູ່ເບິ່ງແບບງ່າຍໆ, ໂປຣເຈັກນີ້ແຫຼະຄືຄຳຕອບ!
ວິທີໃຊ້ກໍງ່າຍສຸດໆ: ພຽງແຕ່ວາງລິ້ງວິດີໂອລົງໄປ, ເລືອກຄວາມຄົມຊັດທີ່ຢາກໄດ້, ແລ້ວກົດປຸ່ມດາວໂຫລດ! ສ່ຳນີ້ກໍຮຽບຮ້ອຍແລ້ວ.
ມີຫຍັງເດັດແດ່? (ຟັງຊັນທີ່ໜ້າສົນໃຈ)
ໃຊ້ງານງ່າຍແບບສຸດໆ: ໜ້າຕາໂປຣແກຣມຖືກອອກແບບມາໃຫ້ບໍ່ສັບສົນ, ໃຜໆ ກໍສາມາດໃຊ້ເປັນໄດ້ທັນທີ.
ຮອງຮັບຫຼາຍແພລັດຟອມ: ບໍ່ວ່າວິດີໂອຈະມາຈາກ YouTube, Facebook, TikTok, ຫຼືເວັບໄຊອື່ນໆ, ໂປຣເຈັກນີ້ກໍຈັດການໃຫ້ໄດ້ໝົດ.
ສະຫຼາດແບບບໍ່ຕ້ອງບອກ: ຖ້າເຮົາ Copy ລິ້ງວິດີໂອໄວ້ໃນ Clipboard, ໂປຣແກຣມຈະກວດພົບ ແລະ ວາງລິ້ງນັ້ນໃຫ້ເຮົາອັດຕະໂນມັດເລີຍ! ບໍ່ຕ້ອງເສຍເວລາວາງເອງ.
ເລືອກຄວາມຄົມຊັດໄດ້ຕາມໃຈ: ຢາກໄດ້ວິດີໂອແບບຊັດແຈ๋ว (Full HD) ເພື່ອເບິ່ງແບບເຕັມອີ່ມ ຫຼື ແບບທຳມະດາເພື່ອປະຢັດພື້ນທີ່, ກໍເລືອກໄດ້ຕາມສະບາຍ.
ເບື້ອງຫຼັງການເຮັດວຽກ (ມັນເຮັດວຽກແນວໃດ?)
ໂປຣເຈັກນີ້ຄືກັບຮ້ານຄ້າອອນລາຍຮ້ານໜຶ່ງທີ່ເຮັດວຽກເປັນທີມ, ເຊິ່ງມີ 3 ສ່ວນສຳຄັນທີ່ປະສານງານກັນ:
Frontend (ໄຟລ໌ index.html):
ອັນນີ້ຄືໜ້າຕາທີ່ເຮົາເຫັນ ແລະ ພົວພັນໂດຍກົງ. ມັນຖືກອອກແບບມາໃຫ້ເບິ່ງງາມໆ ແລະ ເປັນມິດກັບຜູ້ໃຊ້ດ້ວຍ Tailwind CSS.
ເປັນບ່ອນທີ່ເຮົາຈະເຫັນຊ່ອງໃຫ້ວາງລິ້ງວິດີໂອ, ປຸ່ມເລືອກຄວາມຄົມຊັດ, ແລະ ປຸ່ມ "ດາວໂຫລດ" ອັນໃຫຍ່ໆ. ມັນຄືປະຕູໃຫ້ເຮົາເຂົ້າໄປໃຊ້ງານນັ້ນເອງ.
Backend (ໄຟລ໌ app.py):
ສ່ວນນີ້ຄື "ສະໝອງ" ທີ່ເຮັດວຽກໜັກຢູ່ເບື້ອງຫຼັງ, ໂດຍທີ່ເຮົາບໍ່ເຫັນມັນໂດຍກົງ.
ເມື່ອເຮົາກົດປຸ່ມ "ດາວໂຫລດ" ຢູ່ Frontend, ຄຳສັ່ງກໍຈະຖືກສົ່ງມາທີ່ Backend ໂຕນີ້.
Backend ຈະມີ "ພໍ່ຄົວ" ມືອາຊີບ (ເຊິ່ງກໍຄື Library yt-dlp) ທີ່ຈະໄປດາວໂຫລດວິດີໂອຈາກລິ້ງທີ່ເຮົາໃຫ້ໄປ. ພໍ່ຄົວຄົນນີ້ເກັ່ງສຸດໆ, ດາວໂຫລດໄດ້ຈາກເກືອບທຸກເວັບໄຊວິດີໂອເລີຍ!
ຫຼັງຈາກດາວໂຫລດສຳເລັດ, ມັນຈະຈັດການຕັ້ງຊື່ໄຟລ໌ໃຫ້ງາມໆ ແລະ ສົ່ງໄຟລ໌ນັ້ນກັບຄືນມາໃຫ້ເຮົາດາວໂຫລດລົງເຄື່ອງ.
ນອກຈາກນີ້, Backend ກໍຍັງມີໜ້າທີ່ "ທຳຄວາມສະອາດ" ໂດຍການລຶບໄຟລ໌ວິດີໂອເກົ່າໆ ທີ່ດາວໂຫລດໄປແລ້ວອອກຈາກ Server ເພື່ອບໍ່ໃຫ້ເປືອງພື້ນທີ່ເກັບຂໍ້ມູນ.
Deployment (ໄຟລ໌ Dockerfile):
ອັນນີ້ຄື "ແບບແປນ" ຫຼື "ສູດອາຫານລັບ" ທີ່ບອກວິທີກະກຽມທຸກຢ່າງເພື່ອໃຫ້ໂປຣເຈັກນີ້ສາມາດໄປຣັນຢູ່ເທິງ Server ໄດ້, ບໍ່ວ່າຈະເປັນ Server ຢູ່ບ່ອນໃດກໍຕາມ.
Dockerfile ຈະບອກລາຍລະອຽດວ່າຕ້ອງຕິດຕັ້ງເຄື່ອງມືຫຍັງແດ່ (ເຊັ່ນ: ffmpeg ທີ່ສຳຄັນສຳລັບວິດີໂອ), ຕິດຕັ້ງ Library Python ໂຕໃດແດ່, ແລະ ວິທີການຣັນແອັບຂອງເຮົາ.
ເມື່ອເຮົາໃຊ້ "ແບບແປນ" ນີ້, ມັນຈະສ້າງ "ກ່ອງ" (ເອີ້ນວ່າ Docker Container) ທີ່ພ້ອມໃຊ້ງານ, ເຊິ່ງເຮັດໃຫ້ເຮົາສາມາດນຳໂປຣເຈັກນີ້ໄປເປີດບໍລິການເທິງ Cloud Platform ຕ່າງໆ ໄດ້ງ່າຍໆ ເລີຍ.
ຢາກລອງເອົາໄປໃຊ້ ຫຼື ພັດທະນາຕໍ່? (ສຳລັບນັກພັດທະນາ ຫຼື ຄົນທີ່ມັກໂຄ້ດ)
ຖ້າເຈົ້າຢາກລອງເອົາໂປຣເຈັກນີ້ໄປຫຼິ້ນເອງ ຫຼື ຢາກພັດທະນາຕໍ່, ກໍງ່າຍໆ ເລີຍ:
ດາວໂຫລດໂປຣເຈັກນີ້ລົງເຄື່ອງ (Clone ຈາກ GitHub):
git clone [ລົງລິ້ງ GitHub ຂອງທ່ານບ່ອນນີ້]
cd jarnt0-downloads


ຕິດຕັ້ງເຄື່ອງມືທີ່ຈຳເປັນ (Python Libraries):
pip install -r requirements.txt


ລອງຣັນແອັບເບິ່ງ:
python app.py

ຫຼັງຈາກນັ້ນ, ເປີດ Browser ຂອງທ່ານໄປທີ່ http://localhost:5000 ແລ້ວກໍເລີ່ມໃຊ້ງານໄດ້ເລີຍ!
ຂໍ້ຄວນຮູ້ເພີ່ມເຕີມ (ສຳຄັນເດີ້!)
ໂປຣເຈັກນີ້ຍັງເປັນຕົວທົດລອງຢູ່ເດີ້! ຖ້າຈະນຳໄປໃຊ້ງານຈິງຈັງແບບໃຫຍ່ໆ, ອາດຈະຕ້ອງມີການປັບປຸງເລື່ອງການຈັດການຂໍ້ຜິດພາດ, ຄວາມປອດໄພ, ແລະ ປະສິດທິພາບຕື່ມອີກ.
ບາງຄັ້ງການດາວໂຫລດຈາກບາງເວັບໄຊອາດຈະຕ້ອງການ Cookie (ຂໍ້ມູນການເຂົ້າສູ່ລະບົບ) ເພື່ອໃຫ້ yt-dlp ສາມາດເຂົ້າເຖິງວິດີໂອໄດ້.
yt-dlp ເປັນເຄື່ອງມືທີ່ເກັ່ງຫຼາຍ, ມັນສາມາດດາວໂຫລດວິດີໂອໄດ້ຈາກເວັບໄຊທີ່ຫຼາກຫຼາຍ ແລະ ມີທາງເລືອກໃນການດາວໂຫລດທີ່ລະອຽດ.
